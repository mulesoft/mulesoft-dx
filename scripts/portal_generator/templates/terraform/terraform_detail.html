Getting started

  1. Install Terraform CLI (v1.0+)
  2. Configure the provider block with your Anypoint Platform credentials
  3. Define resources in .tf files
  4. Run terraform init, terraform plan, and terraform apply

Authentication

The provider supports authentication via Connected App credentials (client ID and secret) or bearer token. Configure your credentials in the provider block or via environment variables.
